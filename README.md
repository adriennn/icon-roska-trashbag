# icon-roska-trashbag
a trashbag icon for everybody

#Usage

load the stylesheet in the ```<head>```

    <link rel="stylesheet" href="css/trashbag.css">

add the ``` icon-trashabg``` class to your elements

    <i class="icon-trashbag"></i>

Set styles with css

```
.icon-trashbag{
    font-size: 40pt;
    vertical-align: sub;
    color: black;
}
```
